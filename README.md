# Frontend (Vite + React)

## Local setup

1. Copy `.env.example` to `.env`.
2. Set `VITE_API_BASE_URL` to your backend URL.
   - Default local backend URL: `http://127.0.0.1:5001`
3. Fill Firebase values (`VITE_FIREBASE_*`) if you use media uploads.
4. Install dependencies:
   - `npm install`
5. Start the dev server:
   - `npm run dev`

## Default local ports

- Frontend: `http://127.0.0.1:5000`
- Backend API: `http://127.0.0.1:5001`
